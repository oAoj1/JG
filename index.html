O que é linguagens?

html

O HTML(Hypertext Markup Language) é a tecnologia responsável por fazer as marcações principais da estrutura de um site, NÃO é uma linguagem de programação, e sim de marcação. È como se fosse o esqueleto do site

css

O CSS (Cascading Style Sheet) é a tecnologia que acompanha o HTML, responsável por dar estilo ao nosso site, margens, cores, espaçamentos etc. È como se fosse a roupa do site

js

O Java Script é uma das principais tecnologias do mercado atualmente. A linguagem de programação e responsável por dar funcionalidade ao site. È como se fosse o cérebro do site
