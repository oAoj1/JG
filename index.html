O que é linguagens?

html

O HTML(Hypertext Markup Language) é a tecnologia responsável por fazer as marcações principais da estrutura de um site, NÃO é uma linguagem de programação, e sim de marcação. È como se fosse o esqueleto do site

css

O CSS (Cascading Style Sheet) é a tecnologia que acompanha o HTML, responsável por dar estilo e cores ao nosso site, margens, cores, espaçamentos etc. È como se fosse a roupa do site
