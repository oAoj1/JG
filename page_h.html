HTML

Descrição:

O HTML é a linguagem necessária para criar "estrutura óssea" de uma página web, nela são feitos:


  • Textos
  • Títulos
  • Imagens
  • Divisões
  • Vídeos
  • Exportações
  • Links
  • Tabelas

È uma boa recomendação para quem está querendo começar com tecnologia, antes de estudar a lógica de programação pura em si.
